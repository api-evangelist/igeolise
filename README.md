# iGeoliseThis is a repo for managing machine readable artifacts for the [iGeolise APIs](https://www.traveltimeplatform.com/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [iGeolise APIs](https://www.traveltimeplatform.com/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](https://www.traveltimeplatform.com/)- [Developer](https://www.traveltimeplatform.com/)- [Twitter](https://twitter.com/igeolise)The [APIs.json](https://github.com/api-evangelist/igeolise/blob/master/apis.json) index provides links to all the relevant links for the iGeolise APIs, including the OpenAPI and Postman if they are present.
